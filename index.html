ABOUT HAN

안녕하세요! 웹 퍼블리셔 한초롱입니다.

코딩의 매력에 빠져 공부하고 노력하는 1년 7개월차 웹 퍼블리셔입니다.

모든 사람이 쓰기 쉽고 편리한 사이트를 만들기 위해 웹 접근성과 웹 표준 마크업 자연스러운 반응형 작업을 하는 것이 저의 일이라고 생각합니다.

디자이너 경력이 있어 여러 툴을 사용해 보았기 때문에 손이 빠릅니다. 그리고 다년간의 사회 경험으로 책임감 강합니다.

빠르게 변화하는 웹 세상에 도태되지 않기 위해 노력하고 더 나은 코드 작성을 위해 고민하고 공부하는 웹 퍼블리셔입니다.
